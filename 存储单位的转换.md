# 存储单位之间的互相转换

-   b：表示 bit,位(b)
-   B：表示 Byte,字节(B), 1B = 1 字节 = 8bit(比特)
-   KB：1KB = 1024B (KB：千节字，也写作“K”，可读作“K”)
-   MB: 1MB = 1024KB (MB：兆字节，也可写作“M”，读作“兆”)
-   GB：1GB = 1024MB (GB：吉字节，也可写作“G”，读作“G”或“吉”)
-   TB：1TB = 1024GB (TB：万亿字节，读作“TB”或“千吉”)
-   PB：1PB = 1024TB
-   EB：1EB = 1024PB
-   ZB：1ZB = 1024EB
-   YB：1YB = 1024ZB
-   BB：1BB = 1024YB
-   NB：1NB = 1024BB
-   DB：1DB = 1024NB

1GB = 1024MB = 1024 X 1024KB
